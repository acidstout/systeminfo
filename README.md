# SystemInfo
A comprehensive hardware & software report tool with export functionality.

![title screen](screenshot.png)

  
## Features
- Simple, intuitive user-interface.
- Portable. No installation required.
- Export data in a user-friendly form.
- Update manually. Click a button to get the latest version.
- No telemetry. No phoning home. No automatisms.
- Very small footprint. Uses around 4MB RAM.
- Default language is German with automatic fallback to English on non-German systems.

## Usage
Just run SystemInfo.exe to get a detailed summary of your system.

To see information about your TPM chipset, run the tool in an elevated context (e.g., as admin).

## Known issues
If you get a virus warning it is a false positive. Some anti-virus programs are extremely paranoid and see things which are simply not there. I uploaded the binary to VirusTotal, provide a link to the analysis in the release tag, and none of the anti-virus programs had an issue with it. However, that may change anytime.
